## Установка
Установка зависимостей
```bash
npm ci
```
---

## Запуск тестов
Запуск тестов в терминале
```bash
npm run test
```
Запуск тестов в ui
```bash
npm run test:ui
```

---

## Команды разработки
Установка зависимостей
```bash
npm ci
```
Запуск приложения
```bash
npm run dev
```
Сборка приложения
```bash
npm run build
```
Проверка линтером
```bash
npm run lint
```
Исправления линтером
```bash
npm run lint:fix
```
